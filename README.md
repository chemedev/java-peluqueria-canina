## Ejecutar localmente

Clonar el proyecto

```bash
  git clone https://github.com/me-chell/java-peluqueria-canina.git
```

Ir al directorio del proyecto

```bash
  cd java-peluqueria-canina
```

Crear base de datos

```bash
  DB name: `pets`
  Conexión: `mysql://localhost:3306/pets?serverTimezone=UTC`
```

Credenciales base de datos
```bash
  User: `admin`
  Password: `root`
```
